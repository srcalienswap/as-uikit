[View documentation](https://srcalienswap.github.io/as-uikit/)

## Installation

```bash
# With yarn
yarn add @asuikit/hooks @asuikit/core @asuikit/modals

# With npm
npm install @asuikit/hooks @asuikit/core @asuikit/modals
```
